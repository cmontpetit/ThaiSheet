# ThaiSheet Support

Use GitHub Issues for support once the repository is public:

https://github.com/cmontpetit/ThaiSheet/issues

Before opening an issue, please include:

- Your device model.
- Your iOS version.
- ThaiSheet version and build number.
- Steps to reproduce the problem.
- Screenshots or screen recordings when useful.

For App Store review, use this repository URL as the Support URL once the repository is public.
