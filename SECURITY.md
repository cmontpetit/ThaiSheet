# Security Policy

## Supported Versions

Security fixes are handled on the `main` branch until the first public release. After release, supported App Store versions will be listed here.

## Reporting a Vulnerability

Please do not disclose security issues in a public issue.

Use GitHub's private vulnerability reporting if it is enabled. If private reporting is not available, open a minimal public issue asking for a private contact path without including exploit details.

ThaiSheet does not operate backend services. Most security-sensitive issues are expected to involve local data handling, iCloud sync behavior, dependencies, or bundled assets.
